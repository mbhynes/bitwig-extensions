## Summary
- This directory contains the javascript controller extension for the AKAI MPK series
- The scripts were sourced from https://www.akaipro.com/downloads#mpk225

## Deployment
Copy the modified scripts to the `Extentions/` subdirectory of the Bitwig user library:
```
cp -r controller-scripts/* ~/Documents/Bitwig\ Studio/Controller\ Scripts/
```
